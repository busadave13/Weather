# Project Brief: Weather

## Overview
Weather is a test service for mockery - a .NET-based project used for testing and demonstration purposes.

## Repository
- **GitHub:** https://github.com/busadave13/Weather.git
- **Primary Branch:** main

## Technology Stack
- **Language:** C# 9.0+
- **Framework:** ASP.NET Core 9.0+
- **Runtime:** .NET 9.0+
- **Testing:** xUnit, Moq, FluentAssertions
- **Build Tools:** Docker, dotnet CLI
- **CI/CD:** GitHub Actions
- **Infrastructure:** Terraform

## Project Goals
1. Serve as a test service for mockery demonstrations
2. Provide a reference implementation for .NET best practices
3. Demonstrate workflow automation with Cline

- Weather-related service functionality (to be implemented)
